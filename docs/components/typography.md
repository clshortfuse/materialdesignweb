# Typography

Typography components are inline by default (no automatic padding or margins).
Use `gap` or `padding` on layout components to control spacing.

## Components

- `mdw-display`
- `mdw-headline`
- `mdw-title`
- `mdw-body`
- `mdw-label`

## Example

```html
<mdw-box gap=8>
  <mdw-headline size=large>Heading</mdw-headline>
  <mdw-body size=small>Supporting text</mdw-body>
</mdw-box>
```
