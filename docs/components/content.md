# Content & Surfaces

Use content components to structure information and group UI without custom CSS.

## Components

- `mdw-card`
- `mdw-surface`
- `mdw-list` / `mdw-list-item`
- `mdw-divider`
- `mdw-table`

## Example

```html
<mdw-card outlined>
  <mdw-box padding=16 gap=8>
    <mdw-title size=small>Card title</mdw-title>
    <mdw-body size=small>Supporting text</mdw-body>
  </mdw-box>
  <mdw-divider></mdw-divider>
  <mdw-list>
    <mdw-list-item icon=bolt>Item</mdw-list-item>
  </mdw-list>
</mdw-card>
```
